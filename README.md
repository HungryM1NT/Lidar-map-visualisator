# Запуск редактора
```cargo run -r --bin main```

# Управление

W, S - Вперед/Назад
A, D - Влево/Вправо
Q, E - Вниз/Вверх
Вращение колесиком мыши - изменение скорости полета

# Перевод в binary формат
1) Перейти в python_pcd
2) В main.py заменить file_name на необходимый путь
3) Запустить командой ```uv run main.py```
